# ewei-frontend-taro

基于 Taro 的 跨平台的 小屏方案

输出为 H5 + 微信小程序 + APP(RN)

```
node -v v18.7.0
npm -v 8.15.0
yarn -v 1.22.19
taro -v v3.5.5
```

安装

```
yarn
```

本地启动

```
yarn dev:h5
yarn dev:weapp
yarn dev:rn
```

构建

```
yarn build:h5
yarn build:weapp
yarn build:rn
```
